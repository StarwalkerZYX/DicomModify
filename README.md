# DicomModify
A Go program for DICOM modification (annonymization)

# Use "go get" to clone the repo from Github
under any folder, it does not matter
``` bash
go get github.com/StarwalkerZYX/GoTest
```

# Init Go project
```
cd StarwalkerZYX\DicomModify\
go mod init
```
